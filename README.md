# Financial report 

  ## **Technologies:**
    * Html
    * Css/Scss
    * Js
    * React
    * Node (express.js)

  ## **Tools:**
    * Npm
    * Webpack

  ## **How to start?**
    1. Clone repo from gh
    2. Make sure that you have installed node.js, because app use it
    3. Run npm i to install all dependencies
    4. In command line wirte: "npm start"
